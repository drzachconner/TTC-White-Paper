**Step 1: Install weasyprint**


**Step 2: Verify font availability**
```bash
fc-list | grep -i georgia

```
At least one of these must be available. macOS includes both.

**Step 3: Create print CSS stylesheet** at `renditions/ttc-paper.css`

The CSS must include:
- `@page { size: letter; margin: 1in; @bottom-center { content: counter(page); } }` — US Letter, 1-inch margins, page numbers bottom center
- `body { font-family: "Georgia", "Times New Roman", serif; font-size: 11pt; line-height: 1.6; color: #1a1a1a; }` — Professional serif typography
- `h1 { font-size: 20pt; text-align: center; margin-bottom: 0.5em; }` — Title centered
- `h2 { font-size: 16pt; page-break-before: always; margin-top: 0; }` — Major sections start on new page
- `h2:first-of-type { page-break-before: avoid; }` — Executive Summary stays on page after title
- `h3 { font-size: 13pt; }` — Subsections
- `blockquote { font-style: italic; border-left: 3px solid #666; padding-left: 1em; margin-left: 0; }` — Palmer epigraph and other blockquotes
- `table { border-collapse: collapse; width: 100%; margin: 1em 0; font-size: 10pt; }` — Clean tables
- `th, td { border: 1px solid #ccc; padding: 0.4em 0.6em; text-align: left; }` — Table cells
- `th { background-color: #f5f5f5; font-weight: bold; }` — Table headers
- `pre { white-space: pre-wrap; font-size: 0.85em; background-color: #f8f8f8; padding: 0.8em; border-radius: 4px; }` — ASCII diagram fits on page
- `p { orphans: 3; widows: 3; }` — Prevent orphaned lines
- `strong { font-weight: bold; }` and `em { font-style: italic; }` — Emphasis
- `hr { border: none; border-top: 1px solid #ccc; margin: 2em 0; }` — Section dividers
- `a { color: #1a1a1a; text-decoration: none; }` — Links don't show as blue in print
- Title page styling: first `h1` gets extra top margin to push it down the page; subtitle (first `h2` or `h3` after `h1`) gets centered treatment
- Author/date block if passed via pandoc metadata

Do NOT include: color themes, decorative elements, fancy headers, or anything that would make this look like a marketing piece rather than a professional white paper.

**Step 4: Generate PDF**
```bash
pandoc renditions/01_2026-02-03_MASTER_PAPER_COMPLETE.md \
  -f markdown \
  -t html5 \
  --standalone \
  --css=renditions/ttc-paper.css \
  --toc \
  --toc-depth=2 \
  --metadata title="The NeuroSpinal System as the Primary Tone-Setter" \
  --metadata subtitle="Model and Clinical Application of Talsky Tonal Chiropractic" \
  --metadata author="Dr. Zach Conner" \
  --metadata date="2026" \
  -o /tmp/ttc-paper.html

weasyprint /tmp/ttc-paper.html renditions/TTC-White-Paper-Final.pdf
```

**Step 5: Verify PDF**
- Check file exists and is non-zero size
- Check approximate page count (expect 30-50 pages)
- Spot-check that the title page, table of contents, and a few section breaks render correctly

**Pitfall handling from research:**
- ASCII diagram overflow: CSS `pre` uses `white-space: pre-wrap` and smaller font
- Orphaned references: CSS `orphans: 3; widows: 3` prevents single-line orphans
- Font fallback: Verified Georgia/Times New Roman exist before building

**Appendix B handling:** Include Appendix B in the PDF by default. The disclaimer added in Phase 5 notes it "may be omitted in distribution versions" — the author can decide at the checkpoint whether to generate a second PDF without it. Do NOT modify the master markdown to remove it.
